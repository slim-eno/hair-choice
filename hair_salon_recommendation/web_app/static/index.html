关于我们

理发店个性化推荐系统是一个基于位置与社交媒体文本分析的学术研究项目。 我们利用先进的自然语言处理技术，分析社交媒体上的理发店评价， 为用户提供距离优先、好评优先及发型风格优先的分类推荐。

本项目严格限制在学术研究范畴，数据采集采用最小数据集、非商业、匿名化原则。
